# Hospital-SQL-Project 
# مشروع تحليل بيانات المستشفى باستخدام SQL

Project Overview | نبذة عن المشروع

هذا المشروع عبارة عن تحليل شامل لبيانات مستشفى افتراضي باستخدام لغةSQL Server يهدف المشروع إلى الإجابة على مجموعة من الأسئلة التشغيلية والمالية التي تهم الإدارة التنفيذية  مثل تتبع مواعيد المرضى وتقييم أداء الأطباء والأقسام وحساب الإيرادات والقيمة الدائمة للمريض (Patient Lifetime Value) و تصميم استعلامات تدعم لوحات التحكم التنفيذية (Executive Dashboards).

This project presents a comprehensive data analysis for a hospital management system using SQL Server . It tackles complex business questions related to patient journeys, doctor performance, departmental revenue, and financial metrics to support executive decision-making.


 Database Schema | هيكل قاعدة البيانات
 
 المشروع يتعامل مع الجداول التالية المترابطة : 
- Patients (بيانات المرضى)
- Doctors (بيانات الأطباء)
- Departments (الأقسام الطبية)
- DoctorDepartments (ربط الأطباء بالأقسام)
- Appointments (المواعيد والزيارات)
- Treatments (الإجراءات الطبية والتكاليف)
- Prescriptions (الوصفات الطبية)


 Key SQL Concepts Demonstrated 
تم تطبيق مفاهيم متقدمة في هذا المشروع تشمل:
- Multi-table Joins (INNER JOIN , LEFT JOIN) لربط البيانات المعقدة
- Common Table Expressions (CTEs) لتنظيم الاستعلامات الطويلة وحساب المتوسطات
- Window Functions (RANK(), ROW_NUMBER(), LAG(), LEAD(), SUM() OVER()) لتحليل المسارات الزمنية للمرضى وبناء لوحات الصدارة 
- Advanced Aggregations وحساب النسب المئوية للمساهمة في الإيرادات

 Business Questions Addressed | الأسئلة التحليلية التي يجيب عليها المشروع

يتضمن الملف التحليلي حلولاً للاستعلامات التالية:

1. تحليل أداء الأطباء والأقسام :
-  تحديد الطبيب صاحب أعلى متوسط تكلفة علاج.
-  معرفة الأطباء الذين لديهم مواعيد أعلى من المعدل العام.
-  تقرير مفصل لكل طبيب (عدد المرضى، عدد المواعيد، الإيرادات، وترتيبه المالي).
-  تحديد الطبيب الأكثر تحقيقاً للإيرادات في كل قسم طبي على حدة.

2. تحليل سلوك وإنفاق المرضى :
-  معرفة المرضى الذين زاروا أكثر من طبيب.
-  مقارنة تكلفة علاج المريض الفردي بالمتوسط العام لإنفاق المرضى.
-  تتبع التاريخ الزمني لمواعيد المريض (الموعد السابق، الموعد القادم، وأول موعد) باستخدام الدوال الزمنية.
-  تحديد "المرضى المخلصين" (Loyal Patients) الذين زاروا 3 مرات على الأقل وتجاوز إنفاقهم المتوسط.

3. التحليلات المالية والتشغيلية :
-  تحديد الشهر الأكثر ازدحاماً بالمواعيد على مدار السنوات.
-  حساب النسبة المئوية لمساهمة كل مريض في إجمالي إيرادات المستشفى مع ترتيبهم .
-  حساب القيمة الدائمة للمريض .
-  بناء استعلام شامل يغذي لوحة تحكم رئيسية للإدارة الإستراتيجية يضم إجمالي المرضى، الأطباء، المواعيد، العلاجات، الإيرادات، ومتوسط الدخل لكل مريض.
